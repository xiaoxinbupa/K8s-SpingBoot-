# K8s云端发布SpingBoot项目

#### 介绍
这是一个Spingboot项目的部署，使用到nginx，docker，mysql，redis，rabbis，kubernetes，华为云，git，gitee，jenkins，node.js, nfs等技术栈

按顺序构建

1、jumpserver 安装部署

2、harbor 安装部署

3、数据库部署

4、nacos部署

5、RocketMQ服务部署

6、Maven部署

7、jenkins安装和部署

8、 Node部署

9、CCE容器集群购买与部署

10、部署后端程序

11、使用nginx部署前端页面

12、发布服务
